Windows为什么要支持线程

只用一个执行线程的问题在于：长时间运行的任务会阻止其他任务执行。Windows线程的职责是对CPU进行虚拟化，Windows为每个进程都提供了该进程专用的线程(功能相当于一个CPU，可将线程理解成一个逻辑CPU)。


public partial class MainWindow : MainWindow
{
    public MainWindow()
    {
        InitializeComponent();
        Task.Factory.StartNew(() =>
            {
                InvokeMethodExample();
            });
    }

    private void InvokeMethodExample()
    {
        Thread.Sleep(2000);
        Dispatcher.Invoke(() =>
        {
            btn1.Content = "By Invoke";
        });
    }
}
